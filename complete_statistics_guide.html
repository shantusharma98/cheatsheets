1 Descriptive Statistics

What is Descriptive Statistics?

Methods to summarize, organize, and describe data using numerical measures and visualizations. It answers "What is the data?"

📍 Measures of Central Tendency

1. Mean (Average)

Definition: Sum of all values divided by the number of values.

Mean (μ or x̄) = (Sum of all values) / (Number of values) μ = (x₁ + x₂ + ... + xₙ) / n
Example: Scores: 70, 80, 90, 100
Mean = (70 + 80 + 90 + 100) / 4 = 340 / 4 = 85

When to use: Symmetric distributions, no extreme outliers

Sensitive to: Outliers (extreme values pull the mean)

2. Median

Definition: Middle value when data is sorted. 50th percentile.

Odd number of values: Middle value Even number of values: Average of two middle values
Example 1: Scores: 70, 80, 90, 100, 110 (sorted)
Median = 90 (middle value)

Example 2: Scores: 70, 80, 90, 100 (sorted)
Median = (80 + 90) / 2 = 85

When to use: Skewed distributions, presence of outliers

Resistant to: Outliers (not affected by extreme values)

3. Mode

Definition: Most frequently occurring value.

Example: Scores: 70, 80, 80, 90, 100
Mode = 80 (appears twice, most frequent)

Types:

  • Unimodal: One mode
  • Bimodal: Two modes
  • Multimodal: More than two modes

When to use: Categorical data, finding most common value

⚡ Mean vs Median vs Mode:
  • Symmetric distribution: Mean = Median = Mode
  • Right-skewed (positive skew): Mode < Median < Mean
  • Left-skewed (negative skew): Mean < Median < Mode

📏 Measures of Dispersion (Spread)

1. Range

Definition: Difference between maximum and minimum values.

Range = Maximum value - Minimum value
Example: Scores: 70, 80, 90, 100
Range = 100 - 70 = 30

Limitation: Only considers two values, ignores distribution

2. Variance

Definition: Average of squared deviations from the mean.

Population Variance (σ²): σ² = Σ(xᵢ - μ)² / N Sample Variance (s²): s² = Σ(xᵢ - x̄)² / (n - 1) Where: - xᵢ = individual value - μ or x̄ = mean - N = population size - n = sample size - (n-1) = Bessel's correction for unbiased estimator
Example: Scores: 70, 80, 90, 100 (Mean = 85)
Deviations: -15, -5, 5, 15
Squared: 225, 25, 25, 225
Variance = (225 + 25 + 25 + 225) / (4-1) = 500/3 = 166.67

Unit: Squared units (e.g., if data is in dollars, variance is in dollars²)

3. Standard Deviation (SD)

Definition: Square root of variance. Measures average distance from mean.

Population SD (σ): σ = √(σ²) Sample SD (s): s = √(s²)
Example: From above variance = 166.67
SD = √166.67 = 12.91

Interpretation: In normal distribution:

  • 68% of data within ±1 SD from mean
  • 95% of data within ±2 SD from mean
  • 99.7% of data within ±3 SD from mean (Empirical Rule)

4. Interquartile Range (IQR)

Definition: Range of middle 50% of data. Q3 - Q1.

IQR = Q3 - Q1 Where: - Q1 = 25th percentile (First Quartile) - Q3 = 75th percentile (Third Quartile)
Example: Scores: 60, 70, 80, 90, 100, 110, 120
Q1 = 70, Q3 = 110
IQR = 110 - 70 = 40

Use: Resistant to outliers, used in box plots

Outlier Detection:

  • Lower fence: Q1 - 1.5 × IQR
  • Upper fence: Q3 + 1.5 × IQR
  • Values outside fences = outliers

5. Coefficient of Variation (CV)

Definition: Ratio of standard deviation to mean (as percentage). Measures relative variability.

CV = (Standard Deviation / Mean) × 100% CV = (σ / μ) × 100%
Example: Dataset A: Mean = 100, SD = 10
CV = (10/100) × 100% = 10%

Dataset B: Mean = 1000, SD = 50
CV = (50/1000) × 100% = 5%

Interpretation: Dataset B is less variable relative to its mean

Use: Compare variability across datasets with different units or scales

📊 Measures of Position

1. Percentiles

Definition: Value below which a given percentage of data falls.

Example: 90th percentile = 150
Meaning: 90% of values are ≤ 150

Common Percentiles:

  • 25th percentile = Q1 (First Quartile)
  • 50th percentile = Q2 = Median
  • 75th percentile = Q3 (Third Quartile)

2. Quartiles

Definition: Values that divide data into four equal parts.

  • Q1 (25th percentile): 25% of data below this
  • Q2 (50th percentile): Median
  • Q3 (75th percentile): 75% of data below this

3. Z-Score (Standard Score)

Definition: Number of standard deviations a value is from the mean.

Z = (x - μ) / σ Where: - x = value - μ = mean - σ = standard deviation
Example: Score = 90, Mean = 80, SD = 5
Z = (90 - 80) / 5 = 10/5 = 2
Interpretation: Score is 2 standard deviations above mean

Interpretation:

  • Z = 0: Value equals mean
  • Z > 0: Value above mean
  • Z < 0: Value below mean
  • |Z| > 3: Potential outlier

2 Inferential Statistics

What is Inferential Statistics?

Methods to make inferences, predictions, and generalizations about a population based on a sample. It answers "What can we conclude about the population?"

🎯 Key Concepts

Population vs Sample

  • Population (N): Entire group you want to study
    - Parameter: Characteristic of population (μ, σ)
    - Example: All students in a university
  • Sample (n): Subset of population actually studied
    - Statistic: Characteristic of sample (x̄, s)
    - Example: 200 randomly selected students

Sampling Distribution

Definition: Distribution of a sample statistic (like mean) across all possible samples.

Example: Population of 10,000 students
Take 1000 different samples of size 50
Calculate mean for each sample
Plot all 1000 sample means → Sampling distribution

⭐ Central Limit Theorem (CLT) - CRITICAL CONCEPT

Definition: Regardless of the population distribution shape, the sampling distribution of the sample mean will approach a normal distribution as sample size increases.

🎯 The Magic of CLT:

Even if your population is:

  • Skewed (income distribution)
  • Uniform (random numbers)
  • Bimodal (two peaks)
  • ANY shape at all

The distribution of sample means will be approximately normal!

If X₁, X₂, ..., Xₙ are random samples from ANY distribution with: - Mean = μ - Standard Deviation = σ Then the sample mean (x̄) has: - Mean of x̄ = μ (same as population) - Standard Deviation of x̄ = σ/√n (Standard Error) - Distribution: Approximately Normal when n ≥ 30 x̄ ~ N(μ, σ²/n) as n → ∞

  • Standard Error decreases: As n increases, SE = σ/√n gets smaller
🎲 CLT Example 1: Dice Rolls

Population: One die roll (uniform: 1,2,3,4,5,6)
Mean = 3.5, Not normal at all!

Experiment: Roll 30 dice, take average. Repeat 1000 times.

Result: The 1000 averages will form a normal distribution!
- Center around 3.5
- Bell-shaped curve
- SE = σ/√30
💰 CLT Example 2: Income Distribution

Population: Incomes (highly right-skewed)
Few billionaires, many low earners

Sample: Take 50 people, calculate average income
Repeat 500 times

Result: The 500 sample averages → Normal distribution
Even though individual incomes are NOT normal!
💡 Why CLT Matters:
  • Allows us to use normal-based tests (z-test, t-test) even when population isn't normal
  • Justifies confidence intervals for means
  • Foundation of hypothesis testing
  • Makes inference possible with large samples
📝 Interview Question:

Q: "You have exponentially distributed data (not normal). Can you still use a t-test?"

A: "Yes, if sample size is large enough (n ≥ 30). The Central Limit Theorem tells us that the sampling distribution of the mean will be approximately normal, regardless of the population distribution. So even though individual data points are exponential, the sample means will be normally distributed, allowing us to use parametric tests like t-test."
🎯 CLT in Practice:

Standard Error Formula:
SE = σ / √n Examples: - n = 4: SE = σ/2 (half the population SD) - n = 9: SE = σ/3 - n = 25: SE = σ/5 - n = 100: SE = σ/10 As n increases → SE decreases → More precise estimates!

Standard Error (SE)

Definition: Standard deviation of the sampling distribution.

SE of Mean = σ / √n Where: - σ = population standard deviation - n = sample size If σ unknown, use sample SD: SE = s / √n

Interpretation: Measures precision of sample mean as estimate of population mean

Key insight: Larger sample → smaller SE → more precise estimate

  • Not truly random

Non-Probability Sampling Methods

1. Convenience Sampling:

  • Sample whoever is easily accessible
  • Example: Survey people at mall
  • Advantage: Quick, cheap
  • Disadvantage: Highly biased

2. Purposive/Judgmental Sampling:

  • Researcher selects specific individuals
  • Example: Interview industry experts
  • Used in qualitative research

3. Quota Sampling:

  • Like stratified but non-random selection
  • Fill quotas for each subgroup
  • Example: Interview 50 males, 50 females

4. Snowball Sampling:

  • Existing subjects recruit future subjects
  • Example: Study rare disease - patients refer others
  • Good for hard-to-reach populations
⚠️ Non-Probability Sampling Limitations:
  • Cannot calculate sampling error
  • Cannot generalize to population
  • Selection bias
  • No statistical inference possible
💡 Interview Question:

Q: "Which sampling method would you use for a nationwide health survey?"

A: "I'd use stratified cluster sampling:
  • First, stratify by region (North, South, East, West) to ensure geographic representation
  • Then, cluster by city within each region for cost efficiency
  • Finally, randomly sample households within selected cities
  • This balances precision (stratification) with practicality (clustering)"

📏 Confidence Intervals

Confidence Interval (CI):

Range of values that likely contains the true population parameter with a certain level of confidence.

📊 Confidence Interval Concept: 
Population Parameter (μ) - UNKNOWN
           ↓
    ───────▼───────
    ═══════════════  ← True value (we don't know this)

Sample 1:  [─────■─────]     ✓ Contains μ
Sample 2:  [─────■─────]     ✓ Contains μ
Sample 3:      [─────■─────] ✓ Contains μ
Sample 4:  [─────■─────]     ✓ Contains μ
Sample 5:     [─────■─────]  ✗ Misses μ (rare!)
Sample 6:  [─────■─────]     ✓ Contains μ

95% CI means: If we repeat sampling 100 times,
about 95 intervals will contain true μ
⚡ CRITICAL Understanding:

❌ WRONG: "There's 95% probability μ is in this interval"
(μ is fixed, not random!)

✅ CORRECT: "If we repeat this process many times, 95% of intervals will contain μ"
(The interval is random, not μ!)

Confidence Interval for Mean (σ known)

CI = x̄ ± Z(α/2) × (σ / √n) Components: - x̄ = sample mean (center of interval) - Z(α/2) = critical Z-value for confidence level - σ / √n = Standard Error (SE) - Margin of Error (ME) = Z(α/2) × SE Common Confidence Levels: ┌───────────┬──────────┬────────────┐ │Confidence │ α │ Z(α/2) │ ├───────────┼──────────┼────────────┤ │ 90% │ 0.10 │ 1.645 │ │ 95% │ 0.05 │ 1.96 │ │ 99% │ 0.01 │ 2.576 │ └───────────┴──────────┴────────────┘

🔢 Complete Example:

Problem: Sample of 36 students, x̄ = 75, σ = 12
Find 95% CI for population mean

Solution:
Step 1: Calculate SE
SE = σ/√n = 12/√36 = 12/6 = 2

Step 2: Find Z-critical (95% → Z = 1.96)

Step 3: Calculate Margin of Error
ME = 1.96 × 2 = 3.92

Step 4: Build CI
CI = 75 ± 3.92
CI = [71.08, 78.92]

Interpretation: "We are 95% confident that the true population mean lies between 71.08 and 78.92"

Confidence Interval for Mean (σ unknown)

Use t-distribution instead of Z:

CI = x̄ ± t(α/2, df) × (s / √n) Where: - t = t-distribution critical value - df = degrees of freedom = n - 1 - s = sample SD (since σ unknown) Why t-distribution? - When σ unknown, use sample SD (s) - Adds uncertainty → wider intervals - For small samples, t has heavier tails than Z 
t-distribution vs Normal:

  *****         ******
 *******       ********
*********     **********

t has heavier tails (more uncertainty)
As df ↑, t → Z
🔢 Example with Unknown σ:

Problem: n = 16, x̄ = 85, s = 8, find 95% CI

Solution:
Step 1: df = n - 1 = 16 - 1 = 15

Step 2: From t-table: t(0.025, 15) = 2.131
(Compare to Z = 1.96 - t is larger!)

Step 3: SE = s/√n = 8/√16 = 8/4 = 2

Step 4: ME = 2.131 × 2 = 4.262

Step 5: CI = 85 ± 4.262 = [80.738, 89.262]

Interpretation: "We are 95% confident the true mean is between 80.74 and 89.26"

Margin of Error

Definition: Maximum expected difference between sample statistic and population parameter.

Margin of Error (ME) = Critical Value × Standard Error For means (σ known): ME = Z(α/2) × (σ / √n) For means (σ unknown): ME = t(α/2, df) × (s / √n) For proportions: ME = Z(α/2) × √[p(1-p) / n] 
Confidence Interval Structure:

    Lower        Point       Upper
    Bound      Estimate      Bound
      ├───────────┼───────────┤
      │           │           │
      │← Margin →│← Margin →│
      │   of      │   of      │
      │  Error    │  Error    │

CI = Point Estimate ± Margin of Error

Factors Affecting Margin of Error:

  1. Confidence Level:
    ↑ Confidence → ↑ ME (wider interval)
    90% CI: Narrower
    95% CI: Medium
    99% CI: Wider
  2. Sample Size:
    ↑ n → ↓ ME (narrower interval)
    n = 25: ME larger
    n = 100: ME smaller
  3. Variability:
    ↑ σ → ↑ ME (wider interval)
📊 Sample Size Effect:

Same data (x̄=100, σ=15), 95% CI:

n = 9: ME = 1.96×(15/3) = 9.8 → [90.2, 109.8] (wide!)
n = 36: ME = 1.96×(15/6) = 4.9 → [95.1, 104.9]
n = 100: ME = 1.96×(15/10) = 2.94 → [97.06, 102.94]
n = 400: ME = 1.96×(15/20) = 1.47 → [98.53, 101.47] (narrow!)

Pattern: 4× sample size → ½ margin of error
💡 Trade-offs:
  • Higher Confidence ↔ Wider Interval:
    More confident but less precise
  • Lower Confidence ↔ Narrower Interval:
    More precise but less confident
  • Solution: Increase sample size!
    Can have both high confidence AND narrow interval

Confidence Interval for Proportion

CI = p̂ ± Z(α/2) × √[p̂(1-p̂) / n] Where: - p̂ = sample proportion = x/n - x = number of successes - n = sample size Requirements: - np̂ ≥ 10 - n(1-p̂) ≥ 10
🔢 Proportion Example:

Problem: Survey 400 voters, 240 support candidate
Find 95% CI for true proportion

Solution:
Step 1: p̂ = 240/400 = 0.60

Step 2: Check requirements:
np̂ = 400×0.6 = 240 ≥ 10 ✓
n(1-p̂) = 400×0.4 = 160 ≥ 10 ✓

Step 3: SE = √[0.6×0.4/400] = √0.0006 = 0.0245

Step 4: ME = 1.96 × 0.0245 = 0.048

Step 5: CI = 0.60 ± 0.048 = [0.552, 0.648]
Or: [55.2%, 64.8%]

Interpretation: "We are 95% confident that between 55.2% and 64.8% of ALL voters support the candidate"
🎯 Common Interview Questions:

Q1: "Why is a 95% CI wider than a 90% CI?"
A: "To be more confident (95% vs 90%), we need a wider net to catch the true parameter. Higher confidence requires larger critical value (1.96 vs 1.645), which increases margin of error."

Q2: "How does sample size affect CI?"
A: "Larger sample → smaller standard error → narrower CI. Doubling sample size doesn't halve the interval; you need 4× the sample for ½ the width (because SE = σ/√n)."

Q3: "Can you say 95% probability the true mean is in the interval?"
A: "No! The true mean is fixed (though unknown). The INTERVAL is random. Correct interpretation: If we repeated this process 100 times, about 95 of the intervals would contain the true mean."

Null and Alternative Hypotheses

  • Null Hypothesis (H₀): No effect, no difference, status quo
    - Assumes nothing has changed
    - What we try to reject
    - Example: H₀: μ = 100 (mean equals 100)
    - Example: H₀: μ₁ = μ₂ (two means are equal)

    - Claims there IS an effect/difference
    - Can be directional or non-directional
    - Example: H₁: μ ≠ 100 (two-tailed)
    - Example: H₁: μ > 100 (one-tailed, right)
    - Example: H₁: μ < 100 (one-tailed, left)

Significance Level (α)

Definition: Threshold for rejecting H₀. Probability of Type I error we're willing to accept.

Common values:

  • α = 0.05 (5%): Most common standard
    - Accepts 5% chance of false positive
    - 95% confidence level
  • α = 0.01 (1%): More stringent
    - Only 1% chance of false positive
    - Used when false positive is costly
    - Example: Drug approval, medical testing
  • α = 0.10 (10%): More lenient
    - 10% chance of false positive
    - Used in exploratory research

Decision Rule:

If p-value < α → Reject H₀ (statistically significant) If p-value ≥ α → Fail to reject H₀ (not significant) Example with α = 0.05: - p = 0.03 → Reject H₀ (0.03 < 0.05) - p = 0.08 → Fail to reject H₀ (0.08 ≥ 0.05)

Statistical Power (1 - β)

Definition: Probability of correctly rejecting a false null hypothesis. Ability to detect an effect when it exists.

Power = 1 - β Where: - β = Probability of Type II error - Typical target: Power ≥ 0.80 (80%) This means: 80% chance of detecting a real effect

Factors Affecting Power:

  1. Sample Size (n):
    ↑ n → ↑ Power
    Larger sample detects smaller effects
  2. Effect Size:
    ↑ Effect size → ↑ Power
    Bigger differences are easier to detect
  3. Significance Level (α):
    ↑ α → ↑ Power (but ↑ Type I error)
    Trade-off between errors
  4. Variability (σ):
    ↓ σ → ↑ Power
    Less noise makes signal clearer
Power Calculation Example:

Scenario: Test if new drug lowers blood pressure
Expected difference: 10 mmHg
Standard deviation: 15 mmHg
α = 0.05

Sample size needed for 80% power:
n ≈ 36 per group

Interpretation: With 36 people in each group, we have 80% chance of detecting the 10 mmHg difference if it truly exists.
💡 Power Analysis in Practice:
  • Before study: Calculate sample size needed for desired power
  • After study: Calculate achieved power (retrospective)
  • Low power (<50%): Study likely to miss real effects
  • Target power (80%): Good balance
  • High power (>90%): Very likely to detect effect (but may need huge sample)

2. t-Distribution (Student's t)

Definition: Similar to normal but with heavier tails. Used when sample size is small or σ unknown.

Properties:

  • Symmetric around 0
  • Heavier tails than normal (more spread)
  • Shape depends on degrees of freedom (df)
  • As df increases, approaches normal distribution
Degrees of freedom (df) = n - 1 As n → ∞, t-distribution → Normal distribution
When to use:
  • Sample size < 30
  • Population SD unknown (use sample SD)
  • Constructing confidence intervals for mean
  • t-tests

3. Chi-Square Distribution (χ²)

Definition: Distribution of sum of squared standard normal variables.

Properties:

  • Always positive (≥ 0)
  • Right-skewed
  • Shape depends on degrees of freedom
  • As df increases, becomes more symmetric
If Z₁, Z₂, ..., Zₖ are standard normal: χ² = Z₁² + Z₂² + ... + Zₖ² df = k Mean = df Variance = 2 × df
Uses:
  • Chi-square test of independence
  • Goodness of fit test
  • Testing variance

4. F-Distribution

Definition: Ratio of two chi-square distributions. Always positive.

Properties:

  • Right-skewed
  • Always positive
  • Two degrees of freedom (numerator and denominator)
F = (χ₁²/df₁) / (χ₂²/df₂)
Uses:
  • ANOVA (comparing multiple means)
  • Testing equality of variances
  • Regression analysis

5. Uniform Distribution

Definition: All values equally likely in a range.

For continuous uniform on [a, b]: f(x) = 1/(b-a) for a ≤ x ≤ b = 0 otherwise Mean = (a + b) / 2 Variance = (b - a)² / 12
Example: Random number between 0 and 10
All values equally likely
P(X ≤ 5) = 5/10 = 0.5

Real-world: Random number generators, arrival times within interval

6. Exponential Distribution

Definition: Time between events in Poisson process.

f(x) = λe^(-λx) for x ≥ 0 Where λ = rate parameter Mean = 1/λ Variance = 1/λ²
Example: Time between customer arrivals
If average rate = 5 per hour (λ = 5)
Average time between arrivals = 1/5 = 0.2 hours = 12 minutes

Real-world: Time between failures, waiting times, service times

5 Statistical Tests - Parametric

Parametric Tests:

Statistical tests that assume data follows certain distributions (usually normal). More powerful when assumptions are met.

Assumptions: Normality, homogeneity of variance, independence

🔬 t-Tests

1. One-Sample t-Test

Purpose: Compare sample mean to known population mean.

Hypotheses:

  • H₀: μ = μ₀ (sample mean equals population mean)
  • H₁: μ ≠ μ₀ (two-tailed) or μ > μ₀ or μ < μ₀ (one-tailed)
t = (x̄ - μ₀) / (s / √n) Where: - x̄ = sample mean - μ₀ = hypothesized population mean - s = sample standard deviation - n = sample size df = n - 1
Example: Test if average student score = 75
Sample: n=25, x̄=78, s=10
H₀: μ = 75
t = (78-75)/(10/√25) = 3/(10/5) = 3/2 = 1.5
df = 24
Compare to critical t-value or find p-value

When to use: One group, compare to known value

2. Independent Two-Sample t-Test

Purpose: Compare means of two independent groups.

Hypotheses:

  • H₀: μ₁ = μ₂ (no difference between groups)
  • H₁: μ₁ ≠ μ₂
t = (x̄₁ - x̄₂) / √(s₁²/n₁ + s₂²/n₂) Or with pooled variance: t = (x̄₁ - x̄₂) / (sp × √(1/n₁ + 1/n₂)) Where sp² = ((n₁-1)s₁² + (n₂-1)s₂²) / (n₁ + n₂ - 2) df = n₁ + n₂ - 2
Example: Compare test scores: Male vs Female students
Males: n₁=30, x̄₁=75, s₁=8
Females: n₂=35, x̄₂=80, s₂=10
H₀: No difference in average scores

Assumptions: Independence, normality, equal variances (or use Welch's t-test)

3. Paired t-Test (Dependent)

Purpose: Compare means of same group at two different times or conditions.

Hypotheses:

  • H₀: μd = 0 (no difference)
  • H₁: μd ≠ 0
t = (d̄) / (sd / √n) Where: - d̄ = mean of differences - sd = standard deviation of differences - n = number of pairs df = n - 1
Example: Weight before and after diet
Measure each person twice
Calculate difference for each person
Test if average difference ≠ 0

Real-world: Before/after studies, matched pairs

📊 ANOVA (Analysis of Variance)

1. One-Way ANOVA

Purpose: Compare means of three or more independent groups.

Hypotheses:

  • H₀: μ₁ = μ₂ = μ₃ = ... (all means equal)
  • H₁: At least one mean is different
F = MSB / MSW Where: MSB = Mean Square Between groups = SSB / (k-1) MSW = Mean Square Within groups = SSW / (N-k) SSB = Σnᵢ(x̄ᵢ - x̄)² (between-group variation) SSW = ΣΣ(xᵢⱼ - x̄ᵢ)² (within-group variation) df₁ = k - 1 (numerator) df₂ = N - k (denominator) k = number of groups N = total sample size
Example: Compare average sales across 4 regions
Region A: 100, 110, 105
Region B: 120, 115, 125
Region C: 95, 100, 90
Region D: 130, 135, 140

H₀: All regions have same average sales

Post-hoc tests: If F is significant, use Tukey HSD, Bonferroni to find which pairs differ

2. Two-Way ANOVA

Purpose: Examine effects of two independent variables (factors) on dependent variable.

Tests:

  • Main effect of Factor A
  • Main effect of Factor B
  • Interaction effect (A × B)
Example: Effect of Teaching Method (Factor A) and Class Size (Factor B) on test scores
Can detect:
- Effect of method alone
- Effect of class size alone
- Whether effect of method depends on class size (interaction)

3. Repeated Measures ANOVA

Purpose: Compare means across three or more time points for same subjects.

Example: Blood pressure measured at baseline, 1 month, 3 months, 6 months
Same patients measured repeatedly

📈 Correlation and Regression

Pearson Correlation (r)

Purpose: Measure strength and direction of linear relationship between two continuous variables.

r = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / √[Σ(xᵢ - x̄)² × Σ(yᵢ - ȳ)²] Range: -1 ≤ r ≤ 1 r = 1: Perfect positive correlation r = -1: Perfect negative correlation r = 0: No linear correlation

Interpretation:

  • 0.00-0.19: Very weak
  • 0.20-0.39: Weak
  • 0.40-0.59: Moderate
  • 0.60-0.79: Strong
  • 0.80-1.00: Very strong
Example: Correlation between study hours and test scores
r = 0.75 → Strong positive correlation
More study hours associated with higher scores

Assumptions: Linearity, normality, homoscedasticity

Simple Linear Regression

Purpose: Predict one variable (Y) from another (X).

Y = a + bX + ε Where: - Y = dependent variable (outcome) - X = independent variable (predictor) - a = intercept - b = slope - ε = error term Slope (b) = r × (sy / sx) Intercept (a) = ȳ - b × x̄
Example: Predict salary from years of experience
Salary = 30000 + 5000 × Years
Intercept (30000) = starting salary
Slope (5000) = increase per year

R²: Proportion of variance in Y explained by X (0 to 1)

Multiple Linear Regression

Purpose: Predict Y from multiple predictors (X₁, X₂, ...).

Y = a + b₁X₁ + b₂X₂ + ... + bₖXₖ + ε
Example: Predict house price from:
Price = 50000 + 100×SqFt + 20000×Bedrooms + 15000×Bathrooms

🎯 Z-Test

Z-Test for Mean

Purpose: Compare sample mean to population mean when σ is known.

Z = (x̄ - μ₀) / (σ / √n)

When to use:

  • Large sample (n ≥ 30)
  • Population SD (σ) is known
  • Population is normally distributed
Example: National average IQ = 100, σ = 15
Sample of 50 students: x̄ = 105
Is this significantly different?

Z-Test for Proportions

Purpose: Compare sample proportion to population proportion.

Z = (p̂ - p₀) / √(p₀(1-p₀)/n) Where: - p̂ = sample proportion - p₀ = hypothesized proportion
Example: Manufacturer claims 95% pass rate
Sample 200 items: 180 pass (p̂ = 0.90)
Test if actual rate = 95%

6 Statistical Tests - Non-Parametric

Non-Parametric Tests:

Distribution-free tests that don't assume normality. Use when parametric assumptions are violated or data is ordinal/ranked.

📊 Common Non-Parametric Tests

1. Mann-Whitney U Test (Wilcoxon Rank-Sum)

Purpose: Non-parametric alternative to independent two-sample t-test.

Compares: Medians of two independent groups

When to use:
  • Two independent groups
  • Data is ordinal or non-normal
  • Small sample size
Example: Compare customer satisfaction (1-5 scale) between two stores

Parametric equivalent: Independent t-test

2. Wilcoxon Signed-Rank Test

Purpose: Non-parametric alternative to paired t-test.

Compares: Two related samples (before/after, matched pairs)

When to use:
  • Paired/matched data
  • Non-normal differences
  • Ordinal data
Example: Pain rating before and after treatment

Parametric equivalent: Paired t-test

3. Kruskal-Wallis Test

Purpose: Non-parametric alternative to one-way ANOVA.

Compares: Medians of three or more independent groups

H = [12 / (N(N+1))] × Σ(Rᵢ² / nᵢ) - 3(N+1) Where: - N = total sample size - Rᵢ = sum of ranks for group i - nᵢ = sample size of group i
Example: Compare median income across 4 education levels
Groups: High School, Bachelor's, Master's, PhD

Parametric equivalent: One-way ANOVA

4. Friedman Test

Purpose: Non-parametric alternative to repeated measures ANOVA.

Compares: Three or more related groups

Example: Pain ratings at 4 time points (same patients)
Baseline, Week 1, Week 2, Week 3

Parametric equivalent: Repeated measures ANOVA

5. Spearman's Rank Correlation (ρ)

Purpose: Non-parametric alternative to Pearson correlation.

Measures: Monotonic relationship (not just linear)

ρ = 1 - (6Σd²) / (n(n²-1)) Where: - d = difference between ranks - n = number of pairs
When to use:
  • Non-linear monotonic relationship
  • Ordinal data
  • Outliers present
Example: Relationship between class rank and GPA

Parametric equivalent: Pearson correlation

6. Chi-Square Test of Independence

Purpose: Test relationship between two categorical variables.

χ² = Σ[(Oᵢ - Eᵢ)² / Eᵢ] Where: - Oᵢ = observed frequency - Eᵢ = expected frequency - Eᵢ = (row total × column total) / grand total df = (rows - 1) × (columns - 1)
Example: Is gender related to product preference?
Product A Product B
Male 60 40
Female 30 70

7. Chi-Square Goodness of Fit

Purpose: Test if observed distribution matches expected distribution.

Example: Fair die should show each number 1/6 of time
Roll 600 times:
Observed: 1(90), 2(110), 3(95), 4(100), 5(105), 6(100)
Expected: Each should appear 100 times
Test if die is fair

8. Sign Test

Purpose: Simple test for paired data using only direction of change.

Example: Before/after treatment
Count: How many improved (+), how many worsened (-)
Test if significantly more improved than worsened

Simpler than: Wilcoxon signed-rank (doesn't use magnitude)

✅ Assumption Checking

Parametric Test Assumptions:
  1. Normality: Data follows normal distribution
    - Check: Histogram, Q-Q plot, Shapiro-Wilk test
    - If violated & n < 30: Use non-parametric
    - If violated & n ≥ 30: CLT applies, parametric OK
  2. Independence: Observations are independent
    - Check: Study design (random sampling, no pairing)
  3. Homogeneity of Variance: Equal variances across groups
    - Check: Levene's test, F-test
    - If violated: Use Welch's t-test or non-parametric

📊 Decision Tree

Step 1: What type of data?

  • Continuous/Interval: → Parametric or non-parametric
  • Ordinal: → Non-parametric
  • Nominal/Categorical: → Chi-square tests

Step 2: How many groups?

  • 1 group: One-sample test
  • 2 groups: Two-sample test (t-test or Mann-Whitney)
  • 3+ groups: ANOVA or Kruskal-Wallis

Step 3: Independent or Related?

  • Independent: Different subjects in each group
  • Related: Same subjects measured multiple times

Step 4: Check Assumptions

  • Normality: Shapiro-Wilk, Q-Q plot
  • Equal variance: Levene's test
  • Sample size: n ≥ 30 for CLT
⚠️ Common Mistakes to Avoid:
  • Using multiple t-tests instead of ANOVA (inflates Type I error)
  • Using parametric tests on non-normal data with small sample
  • Treating ordinal data as continuous
  • Ignoring paired nature of data
  • Not checking assumptions before testing
💡 Quick Reference:
  • Normal + 2 groups: t-test
  • Normal + 3+ groups: ANOVA
  • Non-normal or ordinal + 2 groups: Mann-Whitney
  • Non-normal or ordinal + 3+ groups: Kruskal-Wallis
  • Categorical variables: Chi-square
  • Correlation: Pearson (normal) or Spearman (not normal)

🎯 Key Takeaways for Interviews

  • Descriptive Stats: Summarize data (mean, median, SD, range)
  • Inferential Stats: Make conclusions about population from sample
  • Normal Distribution: Bell curve, 68-95-99.7 rule
  • t-test: Compare means (1 or 2 groups)
  • ANOVA: Compare 3+ group means
  • Chi-square: Test relationships between categorical variables
  • p-value < α: Reject null hypothesis
  • Type I Error: False positive (reject true H₀)
  • Type II Error: False negative (fail to reject false H₀)
  • Check assumptions: Normality, independence, equal variance
