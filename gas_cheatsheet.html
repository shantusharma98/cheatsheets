⚡ GAS Cheatsheet

Google Apps Script = JS that controls all of Google Workspace

Think of it as a remote control for Google Sheets, Gmail, Drive, Forms, Calendar, and Docs. It runs in the cloud on Google's servers, requires zero setup, and triggers automatically. If you know Python, you already understand 90% of the logic.

The 3 things to understand about GAS
☁️

It runs on Google's servers

No local installation. No pip. No virtual environments. You write code inside a browser, hit Run, and Google executes it. Like a Python script but hosted permanently for free.

Cloud-native
🧩

It speaks "Google Workspace"

Every Google app is a built-in library. SpreadsheetApp, GmailApp, DriveApp — these are equivalent to Python's gspread, smtplib, etc. but built-in with no authentication setup.

JS-based
⏰

Triggers = cron jobs

Run functions automatically on a schedule (daily, hourly), on a form submission, when a sheet is edited, or when a file opens. Exactly like Python cron jobs or Windows Task Scheduler but configured inside Google.

Event-driven

Sheets — Read & Write
Read all data from a sheet
GAS JS Python
⬡ Google Apps Script
function readSheet() {
  const ss = SpreadsheetApp
    .getActiveSpreadsheet();
  const sheet = ss.getSheetByName("Sheet1");
  const data = sheet.getDataRange().getValues();
  // data is a 2D array: [[row1col1, row1col2],[row2col1...]]
  data.forEach(row => {
    Logger.log(row[0] + " - " + row[1]);
  });
}
🐍 Python equivalent
import gspread
from google.oauth2 import service_account

# Need auth setup (GAS handles this automatically)
gc = gspread.service_account("key.json")
sh = gc.open("My Spreadsheet")
sheet = sh.worksheet("Sheet1")
data = sheet.get_all_values()
# data is also a 2D list
for row in data:
    print(row[0], "-", row[1])
Write / update data in a sheet
GAS JS Python
⬡ Google Apps Script
function writeData() {
  const sheet = SpreadsheetApp
    .getActiveSheet();

  // Write a single cell (row 1, col 1)
  sheet.getRange(1, 1).setValue("Hello");

  // Write multiple rows at once
  const newData = [["Alice", 90], ["Bob", 85]];
  sheet.getRange(2, 1, 2, 2).setValues(newData);

  // Append a new row at the bottom
  sheet.appendRow(["Charlie", 77, new Date()]);
}

Mail merge — send personalised emails from a sheet
GAS JS Python
⬡ Google Apps Script
function mailMerge() {
  const sheet = SpreadsheetApp
    .getActiveSheet();
  const rows = sheet.getDataRange().getValues();

  for (let i = 1; i < rows.length; i++) {
    const [name, email, company] = rows[i];
    GmailApp.sendEmail(email,
      `Hello ${name}`,
      `Hi ${name}, welcome from ${company}!`
    );
  }
}
🐍 Python equivalent
import pandas as pd
import yagmail

df = pd.read_excel("contacts.xlsx")
yag = yagmail.SMTP("you@gmail.com")

for _, row in df.iterrows():
  yag.send(
    to=row["email"],
    subject=f"Hello {row['name']}",
    contents=f"Hi {row['name']}, welcome!"
  )

Fetch data from an external API
GAS JS Python
⬡ Google Apps Script
function getExchangeRate() {
  const url = "https://api.exchangerate.host/latest?base=USD";
  const response = UrlFetchApp.fetch(url);
  const data = JSON.parse(response.getContentText());
  const inrRate = data.rates.INR;

  // Write the rate into cell A1
  SpreadsheetApp.getActiveSheet()
    .getRange("A1").setValue(inrRate);
}

Triggers (Automation Scheduling)
onEdit — run when cell value changes
GAS JS Python
⬡ Google Apps Script
// Just name this function "onEdit" — GAS calls it automatically
function onEdit(e) {
  const cell = e.range;
  const newValue = e.value;
  const row = cell.getRow();
  const col = cell.getColumn();

  if (col === 3 && newValue === "Done") {
    // Col C changed to "Done" → highlight green
    cell.setBackground("#00ff00");
  }
}
🐍 Python equivalent

Time-based trigger — run daily at 9am
GAS JS Python
⬡ Google Apps Script
// Run this ONCE to set up the daily trigger
function createDailyTrigger() {
  ScriptApp.newTrigger("sendDailyReport")
    .timeBased()
    .everyDays(1)
    .atHour(9)
    .create();
}

function sendDailyReport() {
  // This runs automatically at 9am every day
  GmailApp.sendEmail("boss@co.com",
    "Daily Report", "Here's today's summary.");
}
🐍 Python equivalent

function createClientFolder(clientName) {
  const rootFolder = DriveApp.getRootFolder();
  const newFolder = rootFolder

Here's the honest strategy: use AI to generate 90% of your GAS code, use no-code tools for standard workflows, and only touch code when you absolutely must. This is the actual workflow of smart automation professionals.

🤖

Claude or ChatGPT → Paste into GAS Editor

Describe what you want in plain English. The AI writes the complete working GAS script. You paste it. Done. No understanding required for standard tasks.

  • Best for: generating new scripts, debugging error messages, explaining what existing code does
  • Workflow: Describe task in English → Copy code → Paste in Extensions → Apps Script → Click Run
  • Works for 95% of tasks an Automation Executive handles

The Smart Automation Executive Playbook
Step 1 — Understand the ask Read the task. Identify: what's the trigger? what's the action? what data is involved?
Step 2 — Can Make/Zapier do it? If it's connecting two apps with standard actions, use no-code tools. Faster, more reliable, easier to maintain.
Step 3 — Need GAS? Use AI If it needs GAS (custom logic, complex Sheets manipulation), describe it to Claude and paste the output.
Step 4 — Test + explain Run it, verify output, ask AI to add comments so you can explain it in meetings. You're the orchestrator, not the coder.
